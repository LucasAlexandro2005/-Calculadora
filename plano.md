# Gerenciamento de Calculadora

Escolher operacao:
    Nesta Parte o usuario escolhe uma operacao e o resultado deve ser de acordo com a operacao escolhida:

    Se escolher Soma o resultado deve ser:
        num1 + nm2

        Se escolher Subtracao:
            num1 - num2

            Se escolher Multiplicacao:
                Num1 * Num2

              Se escolher Divisao:
                Num1 / Num2


        Pontos de Atencao:
            O resultado so podera ter numeros entre 0 e 10.

         Se for menor que 0:
            Mostrar que a operacao foi invalida


            Caso coloque letras:
                Mostrar no sistema: Digite Apenas Numeros
